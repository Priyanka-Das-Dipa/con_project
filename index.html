    Identical or similar descriptions on every page of a site aren't
    helpful when individual pages appear in search results.
    Wherever possible, create descriptions that accurately describe the
    specific page. Use site-level descriptions on the main home page or other aggregation pages, and
    use page-level descriptions everywhere else. If you don't have time to
    create a description for every single page, try to prioritize your content; at the very
    least, create a description for the critical URLs like your
    home page and popular pages.
